User Restrict Cancellation

This module allow to control restrict user cancellation for Sale Order, Purchase Order, Customer Invoice and Journal Entries.

Allow cancellation by user

Allow cancel button in customer invoice.

Not allow cancellation by user

No more Cancel button in customer invoice.

No more cancel button in journal entries.

Version : 12.0.1

Released Date : 10-08-2019

Author by : Matrix Developer Group


 Breaking Changes

  • Configuration at Settings > Users > Allow Cancellation.

  • Allow or not cancellation for Customer Invoice.

  • Allow or not cancellation for Customer Payment.

  • Allow or not cancellation for vendor Bill.

  • Allow or not cancellation for Journal Entries.
